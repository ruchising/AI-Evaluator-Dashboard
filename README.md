# AI Evaluator Dashboard

## Project Overview
AI Evaluator Dashboard is a Django-based web application that evaluates AI-generated responses based on key metrics such as accuracy, relevance, and coherence.  
It provides a simple interface where users can input prompts and analyze AI responses using a structured evaluation system.

---

## Features
- Input prompt-based AI response evaluation
- Score generation for:
  - Accuracy
  - Relevance
  - Coherence
- Clean dashboard UI
- Django backend with modular structure
